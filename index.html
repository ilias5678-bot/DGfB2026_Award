📋 Kurzanleitung: Poster Award

Herzlich willkommen zum Begutachtungsprozess! Damit die Bewertung der Poster reibungslos gelingt, beachten Sie bitte folgende Schritte:

1. Anmeldung

Geben Sie Ihren Namen/Kürzel oben im Feld ein. Er wird automatisch Ihrer Jury-ID zugeordnet.

2. Bewertungsvorgang

  • Wählen Sie ein Poster aus der Liste.
  • Nutzen Sie die 3 Kriterien (1-5 Punkte).
  • Zwischenspeichern: Ihre Eingaben werden automatisch lokal gespeichert. Sie können jederzeit pausieren.

3. Abschluss (WICHTIG!)

Sobald Sie fertig sind, klicken Sie ganz unten auf den grünen Button „Zentral Speichern“. Erst dann werden die Daten an die Datenbank übertragen.
